Bot API SuperMessenger

HTTP и WebSocket API для ботов: отправка сообщений, получение входящих обновлений.

Быстрый старт

  1. Создайте бота в мессенджере («Мои боты»).
  2. Сгенерируйте API-токен в настройках бота.
  3. Вызывайте REST или подключите WebSocket с параметрами login и token.

Интерактивный пример с редактором скрипта →

Аутентификация

Все методы: login (slug бота) + token (API-токен) в query string URL. GET или POST.

WebSocket

wss://<host>/ws/bot?login=<bot>&token=<token>

События: connected, message (поле update). Клиент: { "action": "ping" }.

JavaScript

<script src="/docs/supra-bot-api.js"></script>
<script>
const api = new SupraBotApi({ login: 'mybot_bot', token: '...' });
api.connectWebSocket({ onMessage: (m) => console.log(m) });
await api.sendMessage({ userLogin: 'user', text: 'Hi' });
</script>

Ограничения

  • Только незашифрованные исходящие сообщения.
  • Inbox на сервере — 1 сутки.
  • Личный чат по userLogin — только если пользователь уже начал диалог с ботом.
  • Управление группами/каналами — бот должен быть участником с ролью администратора (или author для публикации в канале).
