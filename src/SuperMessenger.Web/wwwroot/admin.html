Контент приложения

Поддерживается HTML (теги p, strong, a, ul, li, h3 и т.д.). Скрипты при показе удаляются.

Показывается на главной странице, пока мессенджер полностью не инициализируется. Пустое поле — сплэш отключён.

Стили сплэша. Фон — в .sm-splash-overlay. Логотип — .sm-splash-logo, подпись с названием — .sm-splash-app-name (атрибут data-app-name). Без логотипа показывается крупный .sm-splash-logo-fallback.

Можно использовать {{appName}} — подставится «Название приложения» из настроек выше.

Тело функции, вызываемой при каждом клике по логотипу. Доступны параметры clickCount (число быстрых нажатий подряд, с паузой не более 1 с) и showNotice(текст, подписьКнопки, колбэк?).

Страница регистрации

Приветственный текст над формой. Логотип, название, подпись и описание настраиваются во вкладке «Брендинг».

Темы и цветовая палитра

Базовые цвета для страниц входа и админки. Для каждой темы мессенджера — полная палитра. HEX выбирается пикером; для rgba введите значение вручную.
