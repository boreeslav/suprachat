# Разметка текста сообщений

Текст обычных сообщений (не `mc-content` JSON с файлами/фото) рендерится на клиенте модулем **`SupraMarkdown`** (`vendor/supra-markdown.bundle.js`, исходник `supra-markdown-entry.cjs`).

Поддерживается **Markdown** (markdown-it) и несколько **специальных тегов** SuperMessenger.  
Ссылки на тот же хост и команды `/send?text=…` обрабатываются отдельно — см. [07-bot-api.md](07-bot-api.md#форматирование-текста-и-внутренние-ссылки) и [01-functionality.md](01-functionality.md#deep-links-публичные-ссылки).

---

## Стандартный Markdown

| Синтаксис | Результат |
|-----------|-----------|
| `**жирный**` | **жирный** |
| `*курсив*` | *курсив* |
| `***жирный курсив***` | ***жирный курсив*** |
| `~~зачёркнутый~~` | ~~зачёркнутый~~ |
| `` `код` `` | моноширинный фрагмент (`.mc-msg-code`) |
| ` ``` … ``` ` | блок кода (`.mc-msg-pre`) |
| `# Заголовок` … `####` | заголовки (уменьшенный размер в пузыре) |
| `- пункт` / `1. пункт` | списки |
| `> цитата` | blockquote |
| `[подпись](https://url)` | ссылка (см. ниже про внутренние URL) |
| таблица `\| col \|` | таблица с горизонтальной прокруткой |

Переносы строк сохраняются (`breaks: true`). Произвольный HTML в тексте **не** разрешён — вывод проходит через DOMPurify.

### Legacy-рендер

Если бандл `SupraMarkdown` недоступен, клиент использует упрощённый парсер (`renderMessengerMessageHtmlLegacy`) с тем же набором inline-разметки и спецтегов.

### Превью без разметки

Для коротких превью (список чатов, свёрнутый закреп, цитаты) разметка снимается функцией `stripMessengerMarkdownPlain` — остаётся plain text.

## Специальный тег `[copy]…[/copy]`

Кликабельный фрагмент для **копирования текста в буфер обмена**. Подсвечен серым фоном (класс `.mc-msg-copy`).

### Синтаксис

```markdown
Код активации: [copy]SM-2024-ABCD[/copy]
```

Внутри тега работает обычная inline-разметка:

```markdown
[copy]**ABC-123**[/copy]
```

Вложенные теги `[copy]` поддерживаются (парсер учитывает вложенность).

### Поведение в UI

| Действие | Результат |
|----------|-----------|
| Клик / тап | Копируется **видимый текст** (без markdown-разметки) |
| Долгое нажатие | Режим выбора сообщений **не** включается |
| Режим выбора сообщений | Клик по copy-тегу **не** переключает выделение строки |
| Мобильное меню сообщения | Copy-тег **не** открывает контекстное меню |

После копирования — краткая подсветка (класс `mc-msg-copy--copied`) и подсказка «Скопировано».

### Пример для бота

```javascript
const text = 'Ваш токен: [copy]' + token + '[/copy]\nНажмите на токен, чтобы скопировать.';
await api.sendMessage({ userLogin: 'ivan', text });
```

## Ссылки в тексте

### Авто-linkify

URL вида `https://…` и «голые» домены (`example.com/path`) превращаются в кликабельные ссылки, если не были оформлены markdown-ссылкой ранее.

### Внутренние ссылки (deep link)

Ссылки на **тот же origin**, что и мессенджер, открывают объект внутри SPA (класс `mc-msg-internal-link`):

| URL | Действие |
|-----|----------|
| `https://host/@login` | Профиль / личный чат |
| `https://host/@group-uuid` | Группа |
| `https://host/@channel-slug` | Канал |
| `https://host/@parentId/branchSlug` | Ветка группы |
| `…?m={messageId}` | Прокрутка к сообщению |

### Ссылки-команды

`https://host/send?text=…` (или параметр `t`) — отправка текста в текущий чат. Класс `mc-msg-send-link`, пунктирное подчёркивание.  
Активны только в сообщениях от **бота** или **админа/создателя группы**. Подробнее: [07-bot-api.md](07-bot-api.md#ссылки-команды-отправка-текста-в-чат).

### Копирование ссылки из сообщения

В меню сообщения (⋯) для сообщений со ссылками доступен пункт **«Копировать ссылку»** — копирует нормализованный URL в буфер (с уведомлением «Ссылка скопирована»).  
При открытии внешней ссылки в диалоге подтверждения есть кнопка **«Копировать»**.

## CSS-классы разметки

| Класс | Назначение |
|-------|------------|
| `mc-msg-text--md` | Контейнер markdown-текста |
| `mc-msg-code` | Inline-код |
| `mc-msg-pre`, `mc-msg-code-block` | Блок кода |
| `mc-msg-table`, `mc-msg-table-scroll` | Таблица |
| `mc-msg-copy` | Copy-тег (серый фон, клик → clipboard) |
| `mc-msg-internal-link` | Внутренняя навигация |
| `mc-msg-send-link` | Ссылка-команда `/send?text=…` |

## Сборка SupraMarkdown

При деплое и вручную:

```powershell
cd src/SuperMessenger.Web/wwwroot/messenger
npm run build:vendor   # или build-webcrypto.ps1
```

Плагин copy-тега: `supra-markdown-entry.cjs` → `vendor/supra-markdown.bundle.js`.
